#  Личный проект 'Круизы в Антарктику'.

* Исполнитель Шибаев Сергей.

* Проект содержит информацию по выбору ближайшего круиза. Выбор происходит на основании   маршрута, порт отправления, продолжительности, стоимости.
Для выбора круиза в разделе 'Ближайшие круизы' выбрать нужную карточку, навести на нее мышку или выбрать с клавиатуры с помощью кнопки tab, нажать на кнопку 'Забронировать'.
* Вы можете заказать тур заранее, заполнить заявку и отправить к нам в офис, нажав на кнопку 'Оставить заявку'.
* Для удобства перемещения по странице, Вы можете воспользоваться меню, которое расположено вверху и внизу страницы.
